Below is a simple temperature conversion app.

Enter a temperature in either the C or F fields, and the temperature will be converted to the other scale.

temperature C

temperature F

The JavaScript code to deal with the values being input in the fields is below.

const tempC = document.getElementById('tempC')
const tempF = document.getElementById('tempF')

tempC.oninput = function () {
  tempF.value = c2f(tempC.value)
}

tempF.oninput = function () {
  tempC.value = f2c(tempF.value)
}

function c2f (c) {
  return Math.round(c * 9/5 + 32)
}

function f2c (f) {
  return Math.round((f - 32) * 5/9)
}
